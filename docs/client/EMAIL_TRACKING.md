## Настройка отслеживания в Gmail

### Метод 1: Gmail Read Receipts (встроенный)

1. В Gmail нажать "Написать"
2. Внизу справа нажать на три точки (⋮)
3. Выбрать "Request read receipt"
4. Отправить email

**Ограничения:** Работает только если получатель использует Gmail и соглашается отправить receipt.

### Метод 2: Email Tracking Extension

**Рекомендуемые расширения:**
- **Mailtrack** (бесплатно, до 200 emails/месяц)
- **Mixmax** (бесплатно, базовые функции)
- **Yesware** (бесплатно, ограниченные функции)

**Установка Mailtrack:**
1. Перейти на https://mailtrack.io
2. Установить расширение для Chrome/Firefox
3. Включить отслеживание в настройках
4. Отправлять emails как обычно - отслеживание автоматическое

### Метод 3: Pixel Tracking (простой)

Добавить невидимый 1x1 пиксель в HTML email (если отправляете HTML).

## Follow-up Schedule

### Автоматические напоминания

**Через 48 часов (если нет ответа):**
- Отправить follow-up email с case study
- Тема: "Re: [оригинальная тема]"
- Содержание: Краткое напоминание + дополнительная ценность

**Через 72 часа (если нет ответа):**
- Позвонить напрямую (если есть телефон)
- Или отправить LinkedIn message

**Через 96 часов (если нет ответа):**
- Финальный follow-up с предложением trial
- Или перейти к следующим 3 MFO

## Метрики для отслеживания

- **Open Rate:** % открытых emails (цель: 40%+)
- **Response Rate:** % ответов (цель: 5%+)
- **Demo Call Rate:** % запланированных calls (цель: 2%+)
- **Conversion Rate:** % подписанных контрактов (цель: 1%+)

## Best Practices

1. **Персонализация:** Каждый email должен быть персонализирован
2. **Краткость:** Email должен быть прочитан за 30 секунд
3. **Value First:** Сначала ценность, потом предложение
4. **Clear CTA:** Одна четкая call-to-action
5. **Timing:** Отправлять во вторник-четверг, 10:00-14:00 (время Астаны)
